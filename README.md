# OpenLease

OpenLease is a fictional property management platform built as a sandbox environment for experimenting with AI-assisted QA workflow tooling and automated test coverage analysis.

## Purpose

This project exists primarily to support development of "WrightAid," an AI-assisted workflow analysis tool designed to identify potential gaps between application changes and automated test coverage.

## Current Features

- Property dashboard
- Unit tracking
- Tenant assignment
- Rent visibility

## Planned Features

- Maintenance request management
- Payment tracking
- Pet rent billing
- Playwright automated testing
- AI-assisted coverage gap analysis

## Tech Stack

- Node.js
- Express
- Playwright
- JavaScript
